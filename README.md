Laboratorio de desarrollo web N.° 7: práctica de componentes de interfaz interactivos y diseño responsivo, aplicando buenas prácticas de frontend. El código del laboratorio vive en la carpeta `Lab007/` y combina código Java con páginas HTML.

## Temas cubiertos

- Componentes de UI interactivos
- Diseño responsivo con CSS Grid / Flexbox
- Manejo de eventos con JavaScript

## Tecnologías

| Tecnología | Rol en el laboratorio |
|---|---|
| Java | Lenguaje principal del repositorio (carpeta `Lab007/`) |
| HTML5 | Estructura de las páginas |
| CSS3 | Estilos, CSS Grid y Flexbox |
| JavaScript | Interactividad y manejo de eventos |

## Requisitos previos

Antes de ejecutar el laboratorio asegúrate de tener instalado:

- [JDK 8 o superior](https://adoptium.net/) — verifica con `java -version`
- Un IDE con soporte para Java (IntelliJ IDEA, Eclipse o NetBeans)
- Un navegador moderno (Chrome, Firefox o Edge)
- [Git](https://git-scm.com/) para clonar el repositorio
- Opcional: [Visual Studio Code](https://code.visualstudio.com/) con la extensión **Live Server** para servir las páginas HTML

## Instalación y ejecución

1. Clona el repositorio y entra a la carpeta:

   ```bash
   git clone https://github.com/NinaDIV/Web-Lab07.git
   cd Web-Lab07
   ```

2. Abre la carpeta `Lab007/` en tu IDE de Java e importa el proyecto para compilar y ejecutar las clases del laboratorio.

3. Para la parte web, abre `index.html` directamente en tu navegador, o desde VS Code usa la extensión **Live Server** (clic derecho → *Open with Live Server*).

4. Si usas Live Server, la página quedará disponible en `http://127.0.0.1:5500` y se recargará automáticamente al guardar cambios.
